| [Integrator](integrator.md) | Developer embedding ibr as library in pipelines/scripts | 9 stories |
| [Maintainer](maintainer.md) | Developer writing/running automated tests for ibr | 6 stories |
| [AI Coding Assistant](ai-coding-assistant.md) | AI agent invoking ibr on behalf of a developer | 7 stories |
| [Automated Workflow](automated-workflow.md) | Scheduled/event-driven process running ibr unattended | 7 stories |
| [Workspace Operator](workspace-operator.md) | Agent or developer running ibr inside a WSM-managed workspace | 3 stories |
